<p>WebAudio-Controls is a library that provides customizable audio control elements such as knobs, sliders, switches, keyboards, and parameters.<br />It integrates and enhances the following external libraries:</p><br /><ul><br /><li><strong>webaudio-knob</strong> by Eiji Kitamura <a href="http://google.com/+agektmr">Website</a></li><br /><li><strong>webaudio-slider</strong> by Ryoya Kawai <a href="https://plus.google.com/108242669191458983485/posts">Google+ Post</a></li><br /><li><strong>webaudio-switch</strong> by Keisuke Ai <a href="http://d.hatena.ne.jp/aike/">Hatena Blog</a></li><br /></ul><br /><p>Integrated and enhanced by g200kg (Tatsuya Shinyagaito) <a href="http://www.g200kg.com/">Website</a></p><br /><p><strong>Modifications by Bruna Guarnieri:</strong></p><br /><ul><br /><li><strong>Knobs (<code>webaudio-knob</code>):</strong> Fully styled rotary controls with customizable colors, sizes, and behavior.</li><br /><li><strong>Sliders (<code>webaudio-slider</code>):</strong> Linear sliders with horizontal and vertical orientation options.</li><br /><li><strong>Switches (<code>webaudio-switch</code>):</strong> Includes toggle, kick, radio, and sequential types with group handling for radio switches.</li><br /><li><strong>Keyboard (<code>webaudio-keyboard</code>):</strong> Simulates a MIDI keyboard with configurable keys, colors, and MIDI integration.</li><br /><li><strong>Parameters (<code>webaudio-param</code>):</strong> Provides numeric input fields with customizable visual styles and MIDI compatibility.</li><br /><li><strong>Rendering:</strong> Draws controls (like sliders and switches) using the <code>&lt;canvas&gt;</code> API for precise visual control.</li><br /></ul><br /><p><strong>License:</strong><br />Licensed under the Apache License, Version 2.0.
