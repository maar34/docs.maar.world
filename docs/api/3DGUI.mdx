### loadAndDisplayModel {#loadAndDisplayModel}

<div>
  <span className="badge badge--info">static</span>
  <span className="badge badge--success">async</span>
</div>

<p>Loads and displays a GLB model within a Three.js scene using provided track data.</p>

**Returns**: Promise&lt;void&gt;
- <ul><br /><li>A promise that resolves once the model is loaded and added to the scene.</li><br /></ul>

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|scene|THREE.Scene|||<p>The Three.js scene to which the model will be added.</p>|
|trackData|object|||<p>The track data containing information about the interplanetary player.</p>|
|trackData.interplanetaryPlayer|object|||<p>The interplanetary player data.</p>|
|trackData.interplanetaryPlayer.modelURL|string|||<p>The URL of the GLB model to load.</p>|

**Throws**: Error

<ul><br /><li>Throws an error if the modelURL is missing or if loading fails.</li><br /></ul>

```javascript
const scene = new THREE.Scene();
const trackData = {
  interplanetaryPlayer: {
    modelURL: 'https://example.com/models/player.glb'
  }
};

loadAndDisplayModel(scene, trackData)
  .then(() => {
    console.log('Model loaded and added to the scene.');
  })
  .catch(error => {
    console.error('Error loading model:', error);
  });
